1. 模拟盘必须连续运行至少20个真实交易日。目前系统已经开始记录，但需要等待真实交易日逐日累计。
2. 创业板交易权限、券商费率和模拟账户只能由账户本人向券商确认。
3. Choice导出的复权选项需要人工确认，代码无法从纯价格字段可靠推断是否前复权。
4. 回测指标是历史研究结果，不是收益承诺；即使机械闸门通过，也不能直接进入实盘。
5. QMT/PTrade属于第13—14章，不在本阶段连接。
