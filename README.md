# AI Stock Trend Following System

这是一个“每日收盘后运行”的 AI 产业市场状态识别器。它下载 AI 龙头股、纳斯达克100、费城半导体指数和美国10年期国债收益率，生成 `AI_SCORE`（0–100），再将分数映射为中际旭创（`300308.SZ`）的目标仓位。

> 本项目是研究工具，不构成投资建议。Yahoo Finance 数据可能延迟、缺失或调整；实盘前应接入可靠行情并进行独立验证。

## 第一版评分模型

| 模块 | 权重 | 计算逻辑 |
|---|---:|---|
| AI 龙头趋势 | 50% | 龙头股平均20日收益率的5年滚动百分位（70%）+ MA60上方股票占比（30%） |
| 半导体景气 | 30% | SOX 20日动量百分位（50%）+ 位于MA60上方（30%）+ MA60上升（20%） |
| 流动性 | 20% | 10Y收益率20日变化的反向百分位（50%）+ NDX位于MA60上方（30%）+ NDX动量（20%） |

滚动百分位仅使用当日及此前数据，不使用未来数据。ARM 上市历史较短，股票池均值和市场宽度会按当天可用成分股动态计算，至少需要3只有效股票。

| AI_SCORE | 状态 | 目标仓位 |
|---:|---|---:|
| `> 70` | 激进做多 | 100% |
| `50–70` | 正常持仓 | 70% |
| `30–50` | 降低仓位 | 30% |
| `< 30` | 空仓 | 0% |

## 安装与运行

需要 Python 3.10 或更高版本。

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
python -m pip install -r requirements.txt
$env:PYTHONPATH="src"
python run.py
```

运行后会生成：

- `outputs/market_state.csv`：每日子分数、总分、状态和目标仓位；
- `outputs/latest_signal.json`：最新交易信号，便于后续程序读取；
- `outputs/backtest.csv`：中际旭创的简易历史回测；
- `outputs/market_dashboard.png`：价格、AI_SCORE 和仓位图。

回测将收盘后得到的信号延后一个交易日执行，并默认收取单边10个基点的仓位变动成本。它是研究基线，并未包含涨跌停、滑点、税费、停牌和实际成交限制。

## 测试

```powershell
$env:PYTHONPATH="src"
python -m pytest -q
```

## 每日自动运行

确认手动运行正常后，可用 Windows 任务计划程序在美股收盘数据更新后执行 `python run.py`。由于中美市场交易时段不同，信号应用于下一交易日的目标仓位。
